Please refer to the ConestogaInn_Administration_Guide.pdf file in the Documents folder for detailed setup information.

Following are quick set up instructions - 
1. Created the database using the HRS_script_2008_R2.sql file in Database_Script
2. Create an ODBC Data Source with name="HRS" and default database HotelReservationSystem.
3. Run the application using Visual studio 2010 developer or later version.
4. To logon to the application use - userid: admin; password: password
